ALFA NEXUS - Herramientas Digitales para la Búsqueda Académica

1. Repositorios institucionales, Scopus, WoS, redes académicas

Repositorios institucionales: permiten acceder a tesis, artículos y trabajos científicos. Ejemplos: Repositorio UNAM, Digital.CSIC.

Scopus: base de datos bibliográfica con resúmenes, citas y análisis. scopus.com

Web of Science (WoS): plataforma de evaluación e impacto de publicaciones científicas. webofscience.com

Redes académicas: permiten compartir publicaciones y colaborar. ResearchGate, Academia.edu

2. VOSviewer, IA para SLR, SLR‑VTM

VOSviewer: crea mapas bibliométricos para revisar literatura. vosviewer.com

IA para SLR: inteligencia artificial aplicada a revisiones sistemáticas. typeset.io

SLR‑VTM: combina minería de texto con visualización para seleccionar literatura. arxiv.org

3. Mendeley, Zotero, ORCID

Mendeley: gestor de referencias y biblioteca digital. mendeley.com

Zotero: gestor de referencias gratuito. zotero.org

ORCID: identificador único para investigadores. orcid.org

4. Educational Data Mining, Learning Analytics

Educational Data Mining: analiza datos de aprendizaje para predecir resultados. educationaldatamining.org

Learning Analytics: interpreta datos educativos para mejorar decisiones. solaresearch.org

5. Scraping, visualización + Metadig

Scraping + visualización: extracción y análisis visual de datos. Beautiful Soup (Python)

Metadig: plataforma para enseñar alfabetización informacional. metadig.metared.org

6. Plataformas virtuales y redes sociales institucionales

Plataformas virtuales: Moodle, Google Classroom, Microsoft Teams permiten clases virtuales, tareas, y seguimiento. Moodle, Google Classroom

Redes sociales institucionales: adaptaciones educativas de plataformas como WhatsApp o Telegram para comunicación educativa.
